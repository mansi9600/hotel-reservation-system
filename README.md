\# Hotel Reservation System



\## Project Overview



Hotel Reservation System is a Spring Boot and PostgreSQL based application that allows hotel room management through REST APIs.



\## Features



\* Add Room

\* View All Rooms

\* Update Room Details

\* Delete Room

\* PostgreSQL Database Integration

\* Spring Data JPA



\## Technologies Used



\* Java

\* Spring Boot

\* PostgreSQL

\* Maven

\* Spring Data JPA

\* Git \& GitHub



\## API Endpoints



\### Get All Rooms



GET /rooms



\### Add Room



POST /rooms



\### Update Room



PUT /rooms/{id}



\### Delete Room



DELETE /rooms/{id}



\## Database



PostgreSQL database is used to store room information.
